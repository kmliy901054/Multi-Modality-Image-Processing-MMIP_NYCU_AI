# HW1 - Multi-Modality Image Processing

Homework 1 project for NYCU AI course.

## Contents

- `src/main.c`: Main source code.
- `data/`: Input image files used by the program.
- `docs/assignment-1.pdf`: Assignment statement.
- `docs/report.doc`: Report document.
- `docs/templates/`: Report templates.
- `archives/data.7z`: Archived data package.
- `raylib-5.5_win64_mingw-w64/`: Raylib headers and runtime libraries.
- `main.exe`: Prebuilt Windows executable.

## Run (Windows)

This folder already includes a prebuilt executable:

- `main.exe`

To run correctly, keep these in the same `HW1/` directory level:

- `main.exe`
- `data/`
- `raylib-5.5_win64_mingw-w64/`

Then run `main.exe`.

## Build Environment (original)

- OS: Windows
- CPU: x86-64
- Toolchain: MinGW-w64
- Raylib: 5.5 (win64 mingw-w64 package)
